= Description ==
Description: A viking influenced theme with authentic runes and earthy tones. Includes Swedish translation.

== Installation ==
1. Unzip `viking.zip` to the `/wp-content/themes/` directory
2. Activate the theme through the 'Appearance' menu in WordPress


== Known Limitations ==
-A one line menu is recommended, instead of using long menus, use sub menus.
Please note that changing the colors and fonts can impact the accessibility of the theme.
If you choose not to have a sidebar, the width of the content will automatically increase.

If you have any questions or suggestions for this theme, please visit http://wordpress.org/support/theme/viking.
